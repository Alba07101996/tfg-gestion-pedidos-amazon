\# Objetivos del proyecto



\## Objetivo general

Desarrollar una aplicación web para gestionar pedidos de Amazon y su seguimiento logístico.



\## Objetivos específicos

\- Registrar pedidos.

\- Asociar productos a cada pedido.

\- Registrar ASN y facturas.

\- Gestionar el seguimiento de envíos.

\- Permitir búsquedas y filtros.

\- Centralizar la información en una sola aplicación.
