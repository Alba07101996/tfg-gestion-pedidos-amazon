\# Requisitos del proyecto



\## Requisitos funcionales

\- El sistema permitirá crear pedidos.

\- El sistema permitirá editar pedidos.

\- El sistema permitirá eliminar pedidos.

\- El sistema permitirá asociar productos a un pedido.

\- El sistema permitirá registrar ASN.

\- El sistema permitirá registrar facturas.

\- El sistema permitirá consultar el estado de los envíos.

\- El sistema permitirá buscar pedidos por distintos criterios.



\## Requisitos no funcionales

\- La aplicación será accesible desde navegador.

\- La información quedará guardada en una base de datos.

\- La interfaz será sencilla y clara.

\- El sistema tendrá una estructura organizada y mantenible.
